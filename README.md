# Employee Satisfaction Survey System (ESSS)
A web-based AI Chatbot survey system developed using PHP, MySQL, and Vanilla CSS.

## Features
- **Dynamic Bot Flow:** Randomized questions to prevent survey fatigue.
- **Admin Dashboard:** Full-screen management of responses and bot questions.
- **Fixed Intro:** Personalized greeting and identity collection before the survey.
- **Scrollable Interface:** Organized configuration panel for better UX.

## Tech Stack
- Frontend: HTML5, CSS3 (Plus Jakarta Sans font)
- Backend: PHP
- Database: MySQL
